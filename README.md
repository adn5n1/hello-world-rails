# Hello World Rails App

This project was created as part of a technical task for a Junior Software Developer apprenticeship.

## Overview
A simple Ruby on Rails application that displays a "Hello World" message and the current server time using ERB.

## Requirements
- Ruby
- Rails
- Bundler

## Running the Application

1. Clone the repository

git clone https://github.com/adn5n1/hello-world-rails.git

2. Navigate into the project

cd hello-world-rails/hello_app

3. Install dependencies

bundle install

4. Start the Rails server

rails server

5. Open the page in your browser

http://localhost:3000/home/index
